## Sinopsis

Esta seccion centraliza las vistas documentadas del frontend movil para que el equipo pueda identificar rapidamente:

- el codigo de cada vista (`V01`, `V02`, `V03`, etc.),
- una referencia visual movil y web,
- y, cuando exista, la epica o historia de usuario a la que pertenece.

<Aside type="note" title="Como mantener esta seccion">
  La galeria ya quedo preparada para crecer. Las imagenes se gestionan desde `public/Views/` y se asignan en `src/data/mobileViews.ts`. Para vistas con multiples capturas, el componente genera automaticamente un mini-carrusel con navegacion por dots.
</Aside>

## Convencion usada

- `Vxx` identifica de forma unica cada vista documentada.
- La relacion con epica e historia de usuario es opcional: si una vista todavia no esta trazada en backlog, puede quedar solo con una nota descriptiva.
- Los previews actuales pueden sustituirse por mockups o screenshots reales cuando esten disponibles.
